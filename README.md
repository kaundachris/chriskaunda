# Chris Kaunda Personal Website

A clean, editorial-style personal website and blog built to showcase writing, profile, and professional identity through a strong front-end design. This project combines content storytelling with a modern static web experience.

## Project Overview

This website serves as a personal digital home for:

- long-form blog entries on finance, life, and reflection
- an about page that communicates professional background and achievements
- a contact/newsletter page for audience engagement
- a consistent visual identity that reinforces personal branding

## Site Highlights

### Content

The homepage presents a curated collection of blog posts as card-based articles, each with:

- a title and publication month
- a short reflective quote
- a clear call-to-action to read the full post

This structure communicates the site’s editorial personality while keeping navigation simple and user-friendly.

### Project Design

The project uses a static multi-page architecture with a consistent layout system across the site:

- a shared visual style across pages
- separated content, styling, and script concerns
- reusable navigation patterns
- clear hierarchy for blog discovery and profile presentation

The design focuses on readability, contrast, and minimalism, supporting an audience-first experience.

### Tools and Stack

This is a frontend web project built with:

- HTML5 for page structure and semantic content
- CSS3 for layout, color, spacing, and visual styling
- Vanilla JavaScript for small interactive enhancements
- Google Tag Manager and Google Analytics for site tracking and measurement

The project is intentionally lightweight and easy to host, making it a strong example of focused front-end craftsmanship without unnecessary framework overhead.

## Design Skills Demonstrated

This repository highlights several frontend and UX design capabilities:

- information architecture for a content-heavy site
- visual consistency across a multi-page experience
- typography and spacing decisions that support readability
- card-based layout design for content discovery
- simple interaction design using lightweight JavaScript
- personal branding through a polished, cohesive visual identity
